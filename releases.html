A single instance of OpenBoxes running v0.6.6

Overview

This is a demonstration of the latest version of OpenBoxes (v0.6.6).

Recent Changes

v0.6.6

  • Allow user to set timestamp on all transactions (consumption, damaged, expired)
  • Improve performance of create transaction feature
  • Allow sysadmin to configure LDAP for authentication / authorizationion
  • Added hostname to the user's session variables (used for debugging purposes)
  • MySQLNonTransientConnectionException: Communications link failure during rollback(). Transaction resolution unknown.
  • Added spring-test dependency required by rendering plugin (NoClassDefFoundError: org/springframework/mock/web/MockHttpServletRequest)

v0.6.5

  • Allow user to override tags in product import feature
  • Allow system to redirect user to details page for a product, shipment, request when a barcode scanner is invoked
  • Calculate QoH at the product group level
  • Dashboard inventory level alerts should use generic product QoH
  • Create new inventory analysis reporting tool based on generic product QoH
  • Include dashboard widget to display most requested items over the past 30 days
  • Allow user to specify a product as "preferred for reorder" for a location and product group
  • Allow user to view QoH over time (monthly QoH graph on stock card)
  • Allow user to update unit price in product import feature
  • Allow admin to disable / configure the email bug report feature
  • Edit Product > Display category tree using user-defined order rather than alphabetical order
  • Problem with dropdown menu on stock card
  • Exception emails should contain more useful subject and should not be sent to emr-requests
  • Improve product import error handling to display more intuitive messages on failed uploads and file format issues
  • Allow user to configure reason codes used in consumption calculation
  • Global Search > Display quantity on hand for each product at the current location
  • Dashboard formatting off: Most requested items
  • InventoryItem | ProductNotFoundException (java.lang.Exception)
  • Derived properties may not be constrained. Property [monthRequested] of domain class org.pih.warehouse.requisition.Requisition will not be checked during validation.

v0.6.4

  • Integration with barcode scanner (beta)
  • Dashboard widget that shows fast-moving items over the past month
  • ETL process to populate inventory periodic snapshot (beta)
  • New graph to show quantity on hand for a single product over time (depends on ETL process above)
